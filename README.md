## Overview
Project created for showing Java skills. 

## Features
- Adding missions.
- Adding rockets.
- Changing rockets and missions statuses.
- Getting description of the current status of the missions.

## Technology Stack
- **Backend**: Java (v21), Spring
- **Tools**: Maven

## How to Build:

Run tests with:
``mvn test``
Build project with:

``mvn install``
